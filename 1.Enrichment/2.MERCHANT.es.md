# Enriquecimiento de Comerciante y Socio

![Enriquecimiento de Datos](../images/data-enrichment.png)

## Le enviaremos una notificación webhook (HTTP POST) en el siguiente formato:

```jsonc
{
	"event": "merchant",
	"payload": {
		"contractDisputeId": "d492bhhnshts7160i570",
		"acquirerMerchantIdentifier": "004000000396144"
	}
}
```

### Campos de la Notificación Webhook

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `event` | string | Sí | Identificador del tipo de evento. Siempre "merchant" para solicitudes de enriquecimiento de comerciante |
| `payload` | object | Sí | Contenedor para los datos de identificación del comerciante |
| `payload.contractDisputeId` | string | Sí | Identificador único interno de Tupi para el contrato de disputa |
| `payload.acquirerMerchantIdentifier` | string | Sí | Identificador único del adquirente para el comerciante |

```jsonc

    "data": {
        "merchant": {

            "legalName": "ACME INC",

                "postalCode": "12929662",
                "street": "Fourth Street",
                "number": "622",

            }

```

### Campos de la Solicitud de Callback

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `type` | string | Sí | Identificador del tipo de solicitud. Siempre "merchant" para datos de comerciante |
| `data` | object | Sí | Contenedor para datos del comerciante y socio |
| `data.merchant` | object | Sí | Objeto de información del comerciante |
| `data.partner` | object | No | Objeto de información del socio (opcional) |

### Campos del Objeto Merchant

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `taxId` | string | Sí | Número de identificación fiscal del comerciante (CNPJ en Brasil) |
| `legalName` | string | Sí | Nombre legal oficial del comerciante |
| `dbaName` | string | Sí | Nombre comercial |
| `acquirerMerchantIdentifier` | string | Sí | Identificador único del adquirente para el comerciante |
| `economicActivityCode` | string | No | Código de clasificación de actividad económica |
| `address` | object | No | Información de dirección del comerciante |
| `contact` | object | Sí | Información de contacto del comerciante |

### Campos del Objeto Partner

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `taxId` | string | Sí | Número de identificación fiscal del socio (CNPJ en Brasil) |
| `legalName` | string | Sí | Nombre legal oficial del socio |
| `dbaName` | string | Sí | Nombre comercial |
| `partnerAcquirerIdentifier` | string | Sí | Identificador único del adquirente para el socio |
| `address` | object | No | Información de dirección del socio |
| `contact` | object | Sí | Información de contacto del socio |

### Campos del Objeto Address

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `country` | string | Sí | Código de país de dos letras (ISO 3166-1 alpha-2) |
| `postalCode` | string | Sí | Código postal |
| `street` | string | Sí | Nombre de la calle |
| `number` | string | Sí | Número de la dirección |
| `complement` | string | No | Información adicional de dirección (edificio, apartamento, etc.) |
| `neighborhood` | string | Sí | Nombre del barrio o distrito |
| `state` | string | Sí | Código del estado o provincia |
| `city` | string | Sí | Nombre de la ciudad |

### Campos del Objeto Contact

| Campo | Tipo | Requerido | Descripción |
|-------|------|----------|-------------|
| `name` | string | Sí | Nombre completo de la persona de contacto |
| `email` | string | Sí | Dirección de correo electrónico de contacto |
| `phone` | string | Sí | Número de teléfono de contacto con código de país |
