Format of the transaction update notification sent by the system.

```ts
type TransactionEvent = {
	event: "transaction",
	payload: {
		contractDisputeId: string,
		transactionIdentifier: string,
		acquirerReferenceNumber: string,
	}
}
```

Expected format for data enrichment.

```ts
type TransactionResponse = {
	transactionIdentifier: string,
	authorizationDate: string,
	authorizationCurrencyCode: string, // ISO 4217
	acquirerReferenceNumber: string,
	authorizationSequentialNumber: string,
	authorizationCodeResponse: string,
	authorizationCode: string,
	cardNumber: string,
	terminalIdentifier: string,
	authorizedAmount: number,
	paymentPlan: number,
	isInternational: boolean,

	// tip: check brand documentation
	authorizationCardDataMethodCode: string,
	authorizationCnpSecurityLevelCode: string,

	installments: [
		{
			acquirerReferenceNumber: string,
			installmentNumber: number,
			installmentDate: string,
			amount: number,
			issuerSettlementStatus: "settled" | "scheduled" | "canceled",
			merchantSettlementStatus: "settled" | "scheduled" | "canceled",
		}
	],
}
```
