# Datos de API – IA Agente – Contracargo

## Agente de OpenAI

![Recomendación de IA](../images/ai-recommendation.png)

Entrada

```json
{
  "language": "es",
  "card_brand": "Visa",
  "reason_code": "13.1 - Merchandise/Services Not Received",
  "transaction": {
    "amount": 29999,
    "date": "2025-01-15T14:30:00Z",
    "merchant_id": "MERCH-12345"
  },
  "chargeback": {
    "amount": 29999,
    "received_date": "2025-02-20T09:15:00Z"
  },
  "dispute_details": {
    "cardholder_claim": "El cliente afirma que el paquete nunca fue entregado a pesar de que el seguimiento muestra confirmación de entrega",
    "merchant_response": "El comerciante refuta la reclamación"
  },
  "documents_available": ["URLs de Documentos"]
}
```

| **Campo**                           | **Tipo**           | **Descripción**                                                    | **Ejemplo**                                                         |
| ----------------------------------- | ------------------ | ------------------------------------------------------------------ | ------------------------------------------------------------------- |
| `language`                          | String (ISO 639-1) | Código de idioma utilizado para textos                             | `"es"`                                                              |
| `card_brand`                        | String             | Red/marca de tarjeta involucrada en la transacción                 | `"Visa"`                                                            |
| `reason_code`                       | String             | Código de motivo y descripción del contracargo según la red        | `"13.1 - Merchandise/Services Not Received"`                        |
| `transaction`                       | Object             | Detalles de la transacción original                                | *Ver subcampos abajo*                                               |
| `transaction.amount`                | Decimal            | Monto de la transacción en la unidad monetaria más pequeña         | `29999` (=$299.99)                                                  |
| `transaction.date`                  | String (ISO 8601)  | Fecha y hora de la transacción original (UTC)                      | `"2025-01-15T14:30:00Z"`                                            |
| `transaction.merchant_id`           | String             | Identificador único del comerciante                                | `"MERCH-12345"`                                                     |
| `chargeback`                        | Object             | Detalles específicos de la disputa                                 | *Ver subcampos abajo*                                               |
| `chargeback.amount`                 | Decimal            | Monto del contracargo                                              | `29999`                                                             |
| `chargeback.received_date`          | String (ISO 8601)  | Fecha y hora en que se recibió el contracargo (UTC)                | `"2025-02-20T09:15:00Z"`                                            |
| `dispute_details`                   | Object             | Información adicional de la disputa                                | *Ver subcampos abajo*                                               |
| `dispute_details.cardholder_claim`  | String             | Descripción de la reclamación realizada por el tarjetahabiente     | `"El cliente afirma que el paquete nunca fue entregado..."`         |
| `dispute_details.merchant_response` | String             | Descripción de la respuesta proporcionada por el comerciante       | `"El comerciante refuta la reclamación"`                            |
| `documents_available`               | String             | URLs de documentos                                                 | *Archivo adjunto*                                                   |
