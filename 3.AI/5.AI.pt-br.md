# Dados da API – IA Agente – Chargeback

## Agente da OpenAI

![Recomendação de IA](../images/ai-recommendation.png)

Entrada

```json
{
  "language": "pt-BR",
  "card_brand": "Visa",
  "reason_code": "13.1 - Merchandise/Services Not Received",
  "transaction": {
    "amount": 29999,
    "date": "2025-01-15T14:30:00Z",
    "merchant_id": "MERCH-12345"
  },
  "chargeback": {
    "amount": 29999,
    "received_date": "2025-02-20T09:15:00Z"
  },
  "dispute_details": {
    "cardholder_claim": "Cliente alega que o pacote nunca foi entregue, apesar do rastreamento mostrar confirmação de entrega",
    "merchant_response": "Comerciante refuta a alegação"
  },
  "documents_available": ["URLs de Documentos"]
}
```

| **Campo**                           | **Tipo**           | **Descrição**                                                     | **Exemplo**                                                       |
| ----------------------------------- | ------------------ | ----------------------------------------------------------------- | ----------------------------------------------------------------- |
| `language`                          | String (ISO 639-1) | Código do idioma usado para textos                                | `"pt-BR"`                                                         |
| `card_brand`                        | String             | Rede/bandeira do cartão envolvida na transação                    | `"Visa"`                                                          |
| `reason_code`                       | String             | Código de motivo e descrição do chargeback conforme a rede       | `"13.1 - Merchandise/Services Not Received"`                      |
| `transaction`                       | Object             | Detalhes da transação original                                    | *Veja subcampos abaixo*                                           |
| `transaction.amount`                | Decimal            | Valor da transação na menor unidade monetária                     | `29999` (=R$299,99)                                               |
| `transaction.date`                  | String (ISO 8601)  | Data e hora da transação original (UTC)                           | `"2025-01-15T14:30:00Z"`                                          |
| `transaction.merchant_id`           | String             | Identificador único do comerciante                                | `"MERCH-12345"`                                                   |
| `chargeback`                        | Object             | Detalhes específicos da disputa                                   | *Veja subcampos abaixo*                                           |
| `chargeback.amount`                 | Decimal            | Valor do chargeback                                               | `29999`                                                           |
| `chargeback.received_date`          | String (ISO 8601)  | Data e hora em que o chargeback foi recebido (UTC)                | `"2025-02-20T09:15:00Z"`                                          |
| `dispute_details`                   | Object             | Informações adicionais da disputa                                 | *Veja subcampos abaixo*                                           |
| `dispute_details.cardholder_claim`  | String             | Descrição da alegação feita pelo portador do cartão               | `"Cliente alega que o pacote nunca foi entregue..."`              |
| `dispute_details.merchant_response` | String             | Descrição da resposta fornecida pelo comerciante                  | `"Comerciante refuta a alegação"`                                 |
| `documents_available`               | String             | URLs de documentos                                                | *Arquivo anexado*                                                 |
